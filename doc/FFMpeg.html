Class: FFMpeg

FFMpeg

A class used to easily encode medias with ffmpeg

Constructor

new FFMpeg()

Constructs a new FFMpeg encoder The constructor will try to find the ffmpeg binary in the same folder as libDuik, or an (ffmpeg) subfolder, or in the scripts folder of After Effects If ffmpeg is not in one of those, the property ffmpegPath will be an empty string, and it has to be set by the user.
Properties:
Name Type Description
ffmpegPath string The path to the FFMpeg binary.
options string The options of the encoder. Default: '-stats'
queue Array An Array of FFMpegQueueItem
